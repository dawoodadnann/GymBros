#GymBros - The Ultimate Gym Management & Workout Tracking App

## Description

GymBros is your gym, but made **actually fun**.  

Designed for gym owners tired of spreadsheets and members who deserve better than a WhatsApp group, GymBros lets you:

- **Manage memberships** effortlessly  
- **Track attendance** and class schedules  
- **Monitor revenue** in real-time  
- **Members get personal dashboards** to log workouts and track Personal Records (PRs)  
- **Leaderboards** to compete with friends  
- **Roast your gym buddies** because why not?  

Built with **modern infrastructure**, GymBros makes managing and enjoying your gym seamless.

---

## 🔧 Tech Stack

| Layer       | Tech |
|------------|-------------------------|
| Backend    | ASP.NET Core 8, EF Core, SQL Server |
| Frontend   | React + Tailwind CSS |
| Auth       | JWT Authentication |
| Database   | SQL Server / PostgreSQL |
| Deploy     | Azure / Railway |

---

## Features

- Full-stack web application with a clean, responsive UI  
- Gym owner dashboard with analytics & membership management  
- Member dashboard for logging workouts and tracking PRs  
- Dynamic leaderboards to see top performers  
- Fun “roast” feature to keep friends competitive  
- Secure JWT-based authentication
